# Domain Glossary

## City
The place a user searches for and views weather for. Covers both what the user types and the specific place weather is fetched for — there is no separate disambiguation step between a typed search and the resolved place.

## Current Conditions
A real-time snapshot of the weather right now for a City. Distinct from the Forecast: Current Conditions describes the present moment, not a prediction.

## Daily Forecast
A single day's predicted weather within a Forecast. A Forecast is composed of five Daily Forecasts.

## Forecast
The set of five Daily Forecasts predicting a City's weather for the next five days. Distinct from Current Conditions, which describes now rather than a future prediction.
